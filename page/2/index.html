关于投资的哲学

我的一些投资的理念

很早之前，我就有个想法，一个东西为什么是如此的，背后是有一套完整的哲学理念去支撑它成为这样子的；所以我一直觉得一个人要懂点艺术，很遗憾我就没有艺术的细胞，所以我很欣赏那些唱歌唱得好的、会画画的、写文章写得很有诗意的人。

​ 我觉得我算是一个很讲究逻辑的人，在我所知道的那些投资的方法中，我觉得价值投资是最有逻辑的，它能够很好地自圆其说，其次，我觉得价值投资是一种艺术，远不算是一种科学；至于科学与艺术的区别，大概可以用是否可以完全一致的重现作为评断的标志：在欧式几何里面，每个三角形的内角和都是180°，这就是科学了，让你画一幅画，每次画出来都不尽相同，这就是艺术了。

​ 偶尔我也会投投机，但是本质上我是相信价值投资的，我是价值投资的信徒。

​ 其次我相信命，特别是当它对我有利的时候，自从我算出来我三十年财运正在途中的时候，尤其如此；有朋友问我，怎么选投资的标的的，怎么赚钱，我说大部分是靠命，我没有说谎，其实我觉得是真的，只是恰好我重仓的那些标的真的很给力，我还是有数不清的亏得不要不要的呢，只是那些仓位大多数都比较小，即使亏损的比例很大也影响不了大局了。

​ 第三，不以涨喜，不以跌悲；for fun下轻注，for money 下重注，我敢下重注的，都是我真心看好的，如果不涨，那就随它了，不强求啊；即使投资的钱都亏没了，我想我的生活应该也没有多大的影响。

​ 其实就是：价值为锚，闲钱投资；尽人事，听天命。己所欲，勿施于人。

论一个时代的终结

“每天你都有机会跟别人擦肩而过，你也许对他一无所知 ，不过也许有一天他会变成你的一个朋友或知己。” 

​ 我与分级基金的相遇是如此，在某个偶然的巧合认识了分级基金，分级基金就如同是我一个多年的好友吧，我们拥有共同的价值观、聊不完的话题，不尬聊。

​ 没有什么意外的话，分级基金将于2020年死亡了；分级基金曾经是我试图建立的一套系统的重要组成部分，是我很多思想的来源，然而它就要死去了；本周，我告知了小伙伴分级基金末日的来临，并把分级B全部卖完了，从此没拖没欠;我想我还是写点什么东西来纪念一下它吧。

​ 我可以一眼看懂分级基金是个金子，我一眼看不懂比特币是个金子；我常常说我一辈子看人都没有错过，只有错过。

Xelnaga

当我第一看到，（折价的）分级A基金的时候，我一秒就能理解它为什么可以是个靠谱的低风险投资的机会，那种感觉就如同两个刚认识的人就是那么投缘，就是如同认识了多年的好友一般。

  • 在某种假设下，可以分级A基金当做永续债券，采用永续债券的方法可以在分级A基金的利率基础上算出更加能够体现分级基金，刚开始学会了分级A基金隐含收益率的近似算法，后来又通过牛顿不动点迭代法可以精确算出分级A的隐含收益率，后来我可以通过程序自动实现分级A的轮动；那个时候我最开心的应该是我的朋友，完全不明白分级基金的人，也相信我跟他们解释的分级A的理论，他们真的是很善解人意的人，知道我就喜欢吹嘘一些很虚的理论，谢谢他们满足了我的虚荣心，我无以为报。
  • 分级A的下折收套利，由于分级基金的上下折机制以及股市的波动，分级基金的下折总会来临，那个时候我还很耐心的给好些人讲过下折的收益的原理，下折份额的如何折算，我不知道他们听懂了没有，没有听懂也没有关系啊，无脑相信我就好了，我就是想卖弄一下啊，哈哈，我对人的要求很高的，不是每个人都有资格无条件相信我的；没有人信仰的神祗是不会有神力的，所以真的很感谢你们愿意喜欢瞎扯鞋有的没的~~
  • 分级基金折价对冲套利，在折价套利最后的黄金年代，XelNaga产生的交易额缴纳的手续费都高得吓人，要知道分级基金的交易佣金是万分之0.X;具体不表了。

​ 其实我的分级基金是半路出家的，知道的有些晚了，不过，懂IT的不一定懂分级基金，懂分级基金的IT不一定懂IT；我写出来的一整套XelNaga可以妥妥的几万F的实盘，我大Perl的运行速度就是秒杀python，因为我们有信仰加成哈；哈哈曾经有段时间我每天下班吃完饭回家就拼命改代码，每天都改到很晚很晚呢~~~

隐含收益率算法：（各个参数的含义我不告诉你哈哈哈）

sub returnForFundA{
    my $p        =    shift;
    my $t        =    shift;
    my $nav        =    shift;
    my $c0        =    shift;
    my $c1        =    shift;
    my $r = 0.20;
    my $rc = 0;
    my ($y1, $y2, $rrc2);
    while(abs($r-$rc)>0.00000001){
        $r = ($r+$rc)/2;
        $rc = ($r/$p) *( ($c0*$t+($nav-1))/((1+$r)**$t) + $c1/$r/((1+$r)**$t));
    } 
    return $r*100;
}

你，有什么梦想？

​ 人生该有不同的阶段。

​ 走到人生的某个阶段时，我决心要成为一个富有之人。这并不是因为爱钱的缘故，而是为了追求那种独立自主的感觉。我喜欢能够自由地说出自己的想法，而不是受到他人意志的左右。–查理芒格
